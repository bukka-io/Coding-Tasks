# Coding-Tasks

## About
It is a project designed to train basic programming skills. You will learn how to:
- create and use functions
- use arithmetic/comparison/assignment/logical operators 
- use built-in flow control instructions
- use enumeration values
- work with optional values 
- define algorithmic complexity 
- use data collections: array, dictionary, set
- work with Objective Oriented Approach
- and many more

## Build and Runtime Requirements
+ Xcode 13.0 or later
+ iOS 13.0 or later
